Guia do Kanban Klippel

Organize tarefas em colunas e cartões, com anotações avulsas ao lado. Tudo é salvo em um arquivo .kanban no seu computador.

Colunas

  • Crie em "Nova coluna", definindo nome e cor.
  • Reordene arrastando o cabeçalho da coluna, ou pelos botões ◀ ▶.

Itens

  • Adicione com título, descrição, nível de urgência e cor.
  • Mova com as setas (cima/baixo, entre colunas) ou arrastando o cartão.
  • Use o botão de descrição para mostrar/ocultar os detalhes.

Arquivo e segurança

  • Ao salvar pela primeira vez, é possível definir uma senha; o arquivo fica criptografado.

Atalhos de teclado

  • Ctrl + N — Novo projeto
  • Ctrl + S — Salvar projeto atual
  • Ctrl + A — Abrir projeto
  • Ctrl + 1 — Exibir ou ocultar anotações
  • Ctrl + 2 — Alternar modo claro/escuro
  • Ctrl + Q — Nova coluna
